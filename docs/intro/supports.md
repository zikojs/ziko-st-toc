# Support & Contribution

## Getting Help
If you run into issues or have questions, you can get support in the following ways:

GitHub Issues: Report bugs or request features at the [GitHub repository](http://github.com/zikojs/ziko-st-toc/issues)

Discussions: Ask questions or share ideas in the [GitHub Discussions](https://github.com/zikojs/ziko-st-toc/discussions)

Streamlit Community: Discuss usage in the [Streamlit community forum](https://discuss.streamlit.io/t/table-of-contents-generator/120992)

## Contributing

**ziko-st-toc** is open-source and contributions are welcome. You can contribute by:

- Reporting bugs or suggesting features via GitHub Issues
- Submitting pull requests to improve functionality or fix issues
- Improving documentation or examples
