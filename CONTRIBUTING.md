# Contributing to ziko-st-toc

First off, thank you for your interest in contributing! 🎉  
By contributing, you help improve **ziko-st-toc**, the interactive Table of Contents component for Streamlit.

---

## How to Contribute

There are several ways to contribute:

1. **Report a Bug**  
   Please check the [issues](https://github.com/zakarialaoui10/ziko-st-toc/issues) first. If it hasn’t been reported, open a new issue with:
   - Steps to reproduce
   - Expected behavior
   - Screenshots or GIFs if applicable

2. **Request a Feature**  
   Open a new issue and label it as `enhancement`. Include:
   - Problem you want to solve
   - Example use case
   - Optional mockup or GIF

3. **Submit a Pull Request (PR)**  
   PRs are welcome! Please follow these steps:

   ```bash
   # 1. Fork the repo
   # 2. Clone your fork
   git clone https://github.com/YOUR_USERNAME/ziko-st-toc.git
   cd ziko-st-toc

   # 3. Install dependencies
   pip install -e .

   # 4. (Optional) Build the frontend
   cd ziko_st_toc/frontend
   npm install
   npm run build

   # 5. Run tests
   pytest
